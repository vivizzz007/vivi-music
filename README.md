## üåü Introduction

**VIVI Music** is not just a music player; it's a statement. Built with the latest **Android 16** standards and **Jetpack Compose**, it offers a fluid, responsive, and deeply customizable audio experience.

Whether you're streaming from **YouTube Music** or playing your local high-res library, VIVI adapts to your style with dynamic **Material You** theming, synced lyrics, and an "Iron-Clad" stable architecture.

## ‚ú® Features

### üé® **Design & Customization**
-   **True Material You**: The UI extracts colors from your album art, creating a unique theme for every song.
-   **Fluid Animations**: Powered by modern Compose transitions for a buttery-smooth feel.
-   **Dark Mode**: Optimized pure black mode for OLED screens.

### üéß **Playback & Audio**
-   **Hybrid Core**: Seamlessly switch between Local Library and YouTube Music streaming.
-   **Gapless Playback**: No interruptions between tracks.
-   **DSP Engine**: Built-in Equalizer, Bass Boost, and Audio Normalization.
-   **Lyrics**: Real-time synced lyrics (Karaoke style) via LrcLib and Musixmatch.

### üöÄ **Power Features**
-   **Offline First**: Cache songs, download playlists, and listen anywhere.
-   **Smart Sharing**: Share deep links or QR codes to instantly sync music with friends.
-   **Android Auto**: Fully optimized driving interface.
-   **Discord RPC**: Show off what you're listening to on your Discord profile.

## üõ†Ô∏è Development

VIVI Music is open-source and built with "Enterprise-Grade" quality in mind.

### Tech Stack
-   **Language**: Kotlin 100%
-   **UI**: Jetpack Compose (Single Activity Architecture)
-   **DI**: Hilt (Dagger)
-   **Async**: Coroutines & Flow
-   **Network**: Retrofit & OkHttp
-   **Image**: Coil
-   **Database**: Room

### Getting Started
1.  **Clone the repo**
    ```bash
    git clone https://github.com/vivizzz007/vivi-music.git
    cd vivi-music
    ```
2.  **Open in Android Studio** (Ladybug or newer recommended).
3.  **Sync Gradle** and run the `app` configuration.
