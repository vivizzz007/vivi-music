Engineered for Performance

Vivi Music kombiniert lokale Musikbibliotheken mit der Power von YouTube Music (Innertube).

Architecture

  • > MVVM Pattern (Jetpack Compose)
  • > Single Activity (MainActivity.kt)
  • > Room Database (MusicDatabase)
  • > Offline-First Strategy

Innertube Client

Direkter API-Zugriff auf `YouTubeRepository`. Streamt Audio in Opus/AAC ohne Werbung. Unterstützt "Mix", "Charts" und "New Releases".

BetterLyrics

Multi-Source Lyrics Engine (`BetterLyricsLyricsProvider`). Fetcht synchrone Texte von LRCLib, Musixmatch und KuGou.

Kizzy RPC

Integrierter Discord Rich Presence Client (`KizzyRPC.kt`). Zeigt Song, Künstler und Album-Cover live in deinem Discord-Profil an.

Smart Cache

`ExoDownloadService` sorgt für nahtloses Caching. Songs werden automatisch gecacht, um Datenvolumen zu sparen.

Install & Use

1

Download APK

Get the latest `universal` build from Releases.

Go to Releases
2

Grant Permissions

On first launch, allow Notification (for playback controls) and Storage (for local file access).

3

Import & Sync

Go to Settings > Content. You can import YouTube Playlists via URL or scan local folders.

Testing & QA

Unit Tests

Testing core logic like `LyricsParser` and `StringUtils`.

./gradlew testDebugUnitTest
UI Tests (Espresso)

Automated UI testing for `SettingsScreen` and Navigation.

./gradlew connectedAndroidTest
Note: Use `vivi-music-testing` branch for latest experimental features.

Contribute Code

Vivi Music is built with **Kotlin** and **Jetpack Compose**. Join us in shaping the future of audio on Android.

Fork & Branch

Clone the repo and create a feature branch from `main`.

Build Debug

Run `./gradlew assembleDebug` to verify your environment.

Pull Request

Submit your PR. Ensure lint checks pass via `.editorconfig`.
